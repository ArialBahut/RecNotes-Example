deployments
Multiple deployment description can be defined.
In the example below:
  • The recnotes example description is selected.
  • The user docum notebook which is in the RecNotes shelve of the my notes library is selected as the source of the static web.
  • The static web will be generated in the target folder.
  • The web site title is Reciprocal notes example.
  • All html images will include the website subdirectory in their path.

In the future, additional information will be needed to transfer static web site to other destination than GitHub. For GitHub, a git command can be used to transfer the files.
