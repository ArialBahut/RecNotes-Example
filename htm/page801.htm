attachment
To attach one or many files to a topic, first create a folder, then use a Windows Explorer window to copy files to the created folder.
WarningMake sure you do not modify the Reciprocal Notes folder structure.

Open folder
On windows, this option opens the windows file explorer in the folder created for attachments to the selected topic. In the example, the 801 number is the id of the attachment topic.
Open file
The open file option is equivalent to double clicking on a file in windows explorer. If an application program is associated with the extension of the file, this application will open the file.
Copy image path to the clipboard
This option can be used to include an image to another page than the current one.
Append image to current page
After using this option, go back to the page to see the image. You might have to refresh the browser page.
